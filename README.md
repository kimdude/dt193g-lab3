# DT193G - Lab 3
## Webbplats med Vue.js

Repot innehåller källkod för webbplatsen [https://k-drama-collection.netlify.app/drama](https://k-drama-collection.netlify.app/drama). Webbplatsen är framtagen med Vue.js och låter besökare lagra och läsa ut dramaserier. [Detta API](https://github.com/kimdude/dt193g-lab2.git) anropas med fetch och draman läses på så vis ut.
